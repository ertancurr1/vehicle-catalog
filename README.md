# Vehicle Catalog System

A PHP OOP learning project demonstrating:

- Abstract classes and inheritance
- Polymorphism (no if-else for vehicle types)
- PSR-4 autoloading
- PDO database operations
- Repository pattern

## Vehicle Types

| Type       | Special Attribute          |
| ---------- | -------------------------- |
| Car        | Fuel Consumption (L/100km) |
| Motorcycle | Engine Capacity (CC)       |
| Truck      | Load Capacity (tons)       |

## Tech Stack

- PHP 8.2
- MySQL/MariaDB
- HTML/CSS/JavaScript (Vanilla)

## Setup

1. Clone the repository
2. Run `composer install`
3. Import `database/schema.sql` into MySQL
4. Configure `config/database.php`
5. Visit `http://localhost/vehicle-catalog/public/`
